# ASN-Network-Website

A simple PHP website for displaying network information for HyberHost ([AS51692](https://as51692.net)) and SacredServers ([AS27498](https://as27498.net)).

## Features
- Responsive design using Tailwind CSS
- Data-driven: loads network info from `data.json`
- Supports both [as51692.net](https://as51692.net) and [as27498.net](https://as27498.net) branding
- Includes datacentres, IXs, BGP communities, and contact info

## Usage
1. Place `index.php` and `data.json` in your web server directory.
2. Edit `data.json` to update network details as needed.
3. Access the site via your web browser.

## Files
- `index.php` — Main PHP site
- `data.json` — Network data (datacentres, IXs, BGP communities)
- `sitemap.xml` — Sitemap for search engines
